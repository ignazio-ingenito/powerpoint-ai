| Path | Contenuto |
|---|---|
| `drafts/` | Repo brief, executive storyline e creative handoff |
| `prompts/` | Prompt usati per generazione o recupero del deck |
| `source-materials/` | Sorgenti e package ricevuti o prodotti per il progetto |
| `visual-references/` | Export slide e reference visive specifiche Cantieri |
| `generated-assets/` | Script e asset generati per il deck |
| `attempts/` | Tentativi, preview e output intermedi |
