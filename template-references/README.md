# Template Visual References

Queste immagini sono estratte da `docs/template.pdf` e servono come anchor visuali per modelli/tool di generazione slide.

Usarle per riprendere:

- font Poppins;
- palette bianco, grigio/nero, teal/azzurro;
- gradienti leggeri e linee organiche del template;
- header/footer, logo e numero pagina;
- card sottili, roadmap, tre colonne e fascia `Valore generato`.

Non usarle come fonte di contenuto cliente.
