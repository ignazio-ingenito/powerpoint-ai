# Research

Area per ricerche durevoli che possono influenzare proposte, business case, tecnologie, vendor, modelli AI, architetture o scelte di piattaforma.

Struttura:

- `active/`: ricerche in corso;
- `accepted/`: ricerche accettate e riusabili;
- `archived/`: ricerche storiche o superate.

Le informazioni temporali o esterne devono essere rivalidate prima di comparire in un deck cliente.
